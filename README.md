# Садиба UA — Розумний помічник українського садівника 🌻

**Садиба UA** — це сучасний веб-додаток (Progressive Web App), створений для допомоги українським садівникам та городникам. Він поєднує науковий підхід агрономії, потужність штучного інтелекту Gemini та надійний офлайн-режим для надання персоналізованих порад щодо догляду за рослинами у будь-яких умовах.

## 🌟 Основні можливості

- 📱 **Встановлення на телефон (PWA):** Додаток можна встановити на головний екран смартфона (Android/iOS) або комп'ютера. Він має власну іконку та працює як повноцінна програма.
- 📶 **Гібридний Офлайн-режим:** Немає інтернету або API-ключа? Не проблема! Додаток містить вбудовану базу знань для найпопулярніших культур та автоматично перемикається на неї. Спеціальні бейджі (✨ ШІ / 📖 Довідник) показують джерело інформації.
- 📍 **Локалізація порад:** Враховує ваш регіон (область та місто) та поточну пору року для надання актуальних завдань на сьогодні (наприклад, попередження про заморозки).
- 🌿 **Мій Сад:** Зручний каталог ваших рослин з детальними довідками, секретами догляду та журналом робіт.
- 🧪 **Розумний аналіз препаратів:** Автоматично розпізнає назви добрив та засобів захисту (фунгіциди, інсектициди) в тексті порад, пропонуючи детальну інструкцію та заходи безпеки при кліку.
- 📄 **Експорт у PDF:** Можливість створити та завантажити професійний звіт про історію догляду за конкретною рослиною.
- 🔒 **Абсолютна приватність:** Усі дані (профіль, список рослин, журнал, API-ключ) зберігаються виключно у вашому браузері (`localStorage`).

## 🛠 Технологічний стек

- **Frontend:** React 19, TypeScript, Vite.
- **PWA:** `vite-plugin-pwa` (Service Worker, Manifest, кешування).
- **Стилізація:** Tailwind CSS, shadcn/ui (Radix UI).
- **Анімації:** Motion (Framer Motion).
- **Штучний інтелект:** Google Gemini API (`@google/genai`).
- **Робота з документами:** jsPDF, html2canvas.
- **Іконки:** Lucide React.

## 🚀 Як запустити проєкт локально

1. **Клонуйте репозиторій:**
   ```bash
   git clone <url-репозиторію>
   cd sadyba-ua
   ```

2. **Встановіть залежності:**
   ```bash
   npm install
   ```

3. **Запустіть додаток у режимі розробки:**
   ```bash
   npm run dev
   ```

4. **Відкрийте в браузері:** `http://localhost:3000`

## 📱 Як встановити додаток на телефон (PWA)

Додаток підтримує встановлення на пристрої:
- **Android (Chrome):** При першому відкритті сайту з'явиться спливаюче вікно "Встановити додаток". Натисніть "Встановити".
- **iOS (Safari):** Натисніть кнопку "Поділитися" (квадрат зі стрілочкою вгору) на нижній панелі Safari та оберіть "На початковий екран" (Add to Home Screen).
- **ПК (Chrome/Edge):** Натисніть на іконку встановлення (монітор зі стрілочкою) у правій частині адресного рядка браузера.

## 🔑 Налаштування ШІ (Gemini API)

Додаток чудово працює в базовому офлайн-режимі, але для отримання унікальних порад від штучного інтелекту потрібен API ключ:

1. Отримайте безкоштовний ключ на [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Відкрийте **Налаштування** (іконка шестерні) у додатку.
3. Вставте ключ у поле **"Власний ключ Gemini API"**.

## 📂 Структура проєкту

- `src/App.tsx` — основна логіка, інтерфейс та PWA-провайдер.
- `src/lib/gemini.ts` — модуль взаємодії з Google Gemini API та логіка перемикання на офлайн-режим.
- `src/lib/staticData.ts` — локальна база знань (офлайн-довідник).
- `src/components/` — багаторазові UI компоненти (shadcn).
- `public/` — статичні ресурси, іконки для PWA (`icon.svg`), маніфест.
